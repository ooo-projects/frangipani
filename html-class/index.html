A short, exploratory course covering the basics of HTML and CSS (or how to create websites 'by hand').

Course structure:
  • Session 1: Observe
  • Session 2: Distill
  • Session 3: Revel
Students will receive a website template, which we will inspect and change during our tutorials.

The tutorials themselves will cover:
  • Basic HTML syntax
  • Basic CSS syntax
  • The Handmade Web: Past and Present
  • Pointers for deeper learning
Our live sessions will take place on Microsoft Teams.

Payments can be made either through Paystack (NGN) or Ko-fi (USD, GBP and others).
